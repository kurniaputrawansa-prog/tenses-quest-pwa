Tenses Quest — Mobile PWA (Upgraded)
Files:
- index.html : Single-file PWA with mobile-app UI, 100-question pool, 20-question rounds, timer, local leaderboard, review, CSV export, confetti.
- manifest.json : PWA manifest
- sw.js : Service worker for offline caching
- icon-192.png, icon-512.png : App icons

How to use:
1. Upload all files to a static hosting (Netlify, Vercel, GitHub Pages, or any static host).
2. Open the site (https) — you should be prompted to install (or use the install button).
3. Works offline after first load.
